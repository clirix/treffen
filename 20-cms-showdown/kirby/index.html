Kirby

Das dateibasierte CMS

WDCM Dresden, 7. Februar 2013

Allgemeine Infos

  • Entwickler: Bastian Allgeier aus Mannheim
  • veröffentlicht im Februar 2010
  • Lizenz: 30 € / Website (in Entwicklungsphase kostenlos)
  • made with Kirby: 6Wunderkinder, Smashing Conference, beyond tellerrand conference, Jessica Hische, …

Vorteile

  • dateibasiert
    • blitzschnelle Installation
    • portabel und leichtgewichtig
    • z.B. mit Dropbox verknüpfbar
  • reduziertes, übersichtliches Setup
  • jQuery-artige API
    • leichter Einstieg
  • nativer Markdown-Support
  • leicht erweiterbar
    • Plugins
    • Kirbytext extensions
  • Mehrsprachigkeit

Nachteile

  • dateibasiert
    • logischerweise nicht für jede Art von Website geeignet
    • aber: leicht mit MySQL verknüpfbar
  • nicht kostenlos
